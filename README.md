# E-commerce Back End

## Purpose 
To build the back end for an e-commerce site using the Express.js API, configure it to use Sequelize to interact with a MySQL database, and test the routes with Insomnia.

## Built With
* JavaScript
* Node.js
* Express.js
* MySQL
* Insomnia
